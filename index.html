Technologies

These are some technologies that I am proficient in.

HTML 5

A markup language used for structuring and presenting content on the World Wide Web.

JavaScript

A high-level, dynamic, untyped, and interpreted programming language. JavaScript is most well known to be a scripting language, supports object-oriented programming styles.

React

React is an open-source JavaScript library for data rendered as HTML. Uses a virtual DOM which allows developers to create applications fast.

Bootstrap

Bootstrap easily and efficiently has templates that you can use for your websites and applications with a single code base, from phones to tablets to desktops with CSS media queries.

NodeJs

Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js' package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

Handlebars

Handlebars is an extension to the Mustache templating language. Handlebars is a templating language that keeps the view and the code separated.

Sequelize

Sequelize is a promise-based Node.js ORM for Postgres, MySQL, SQLite and Microsoft SQL Server. It features solid transaction support, relations, read replication and more.

CSS

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language.

Marvel RPG Game

Select your character and prepare to fight!

Marvel RPG Game is a website game that allows the user to play against the computer. You select a character and an opponent, then attack! Eat Da Burger

Select your character and prepare to fight!

Eat Da Burger is a website application that allows the user to add any type of burger that they want to the burger list. Then you can devour the burger which makes it move from the burger list to the devoured list across the page. This applicaion uses sequelize and handlebars as the main components for development. Also has user authenticaion that is stored in MySQL database.
